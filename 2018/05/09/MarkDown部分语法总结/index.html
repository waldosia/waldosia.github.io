2018-05-09 Hexo, MarkDown Comments

一.hexo的使用

  • 1.1 新建一篇文章

  • 1.2. 生成静态文件

可简写为

  • 1.3. 发表草稿

  • 1.4. 部署服务器。默认情况下，访问网址为： http://localhost:4000/

  • 1.5. 部署网站。

  • 1.6. 在线编写博客 http://localhost:4000/admin

    二. 语法总结

  • 2.1. 换行
    上一句 + 3个以上空格 + enter + 下一句。
  • 2.2. 缩进
    换成全角符号的空格。
  • 2.3. 标题
    多个#号 + 空格 + 标题文字
    #号越多字越小，最多支持6个#号
  • 2.4. 列表
    ·有序列表
    数字. + 空格 + 文本
    ·无序列表
    +/- + 空格 + 文本
  • 2.5. 代码区块（适用于Hexo）
    3个反引号+语言类型（eg.html,css,javaScript）
    代码块
    3个反引号
  • 2.6. 行内代码
    用反引号`包裹住代码即可
  • 2.7. 链接
    [链接文字]+(链接网址)
  • 2.8. 文章中均可直接插入HTML元素
  • 2.9. 利用tabel插入表格时，需要把标签内的空格都去掉。

只列了一部分，具体参考MarkDown
